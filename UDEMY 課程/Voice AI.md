# 各課程詳細說明

## 1. Mastering Voice AI: From ASR to Emotion AI to Voice Cloning

這門是四門裡面最偏「語音 AI 技術本質」的課。

重點包含：

* ASR：Automatic Speech Recognition
* Speech Language Models
* Audio feature extraction
* Speech tokenization
* Transformer
* Emotion Recognition
* Personalized Speech
* Voice Cloning
* WER 評估
* Ethical AI

它比較像：

```text
語音 AI / SpeechLM 研究與工程基礎課
```

適合：

* 想理解 Whisper、Voxtral、SpeechLM 背後原理
* 想做語音情緒辨識
* 想做 Voice Cloning
* 想往語音 AI 研究或工程走

推薦程度：**9 / 10**

---

## 2. Mistral AI: Voxtral TTS, Vision & AI Agents

這門比較新，重點是用 Mistral AI 做實際專案。

內容包含：

* Voxtral real-time STT
* TTS
* Voice cloning
* Vision / OCR
* Function calling
* Multi-agent handoff
* Python API
* Voice assistant
* Live subtitling app
* Homework helper
* Multi-agent personal assistant

它比較像：

```text
Mistral API 實戰課
```

適合：

* 想快速做語音 + 視覺 + Agent 專案
* 想學 Mistral AI 生態系
* 想用免費方案做作品
* 想做 AI 輔助學習工具或語音助理

推薦程度：**8.5 / 10**

---

## 3. Speech Recognition with Python

這門是最適合入門的語音辨識課。

內容包含：

* Speech Recognition 基礎
* Python 語音處理
* Audio Processing
* Speech-to-Text
* Text-to-Speech
* Whisper
* AI Speech Applications

它比較像：

```text
Python 語音辨識入門課
```

適合：

* 還沒做過語音 AI
* 想先學 STT
* 想做簡單轉錄工具
* 想用 Whisper 做作品
* 想建立語音處理基本概念

推薦程度：**8 / 10**

---

## 4. AI Voice Agents: Automation with Vapi, ElevenLabs, n8n & MCP

這門最偏商業應用與自動化。

內容包含：

* Voice Agent 基礎
* Vapi
* ElevenLabs
* LiveKit
* n8n
* MCP
* Webhooks
* RAG
* Phone call agent
* Appointment booking assistant
* Website voice agent
* Outbound call
* Python + LiveKit
* OpenAI Realtime API
* Prompt engineering for voice agents

它比較像：

```text
做可以賣給客戶的 AI 電話客服 / 預約助理
```

適合：

* 想做 AI 語音客服
* 想做預約助理
* 想接 n8n 自動化
* 想整合 Google Calendar / CRM / API
* 想做可商業化的 Voice Agent

推薦程度：**9 / 10**

# 學習順序建議

## 如果你想打好語音 AI 技術基礎

```text
1. Speech Recognition with Python
↓
2. Mastering Voice AI
↓
3. Mistral AI Voxtral
↓
4. AI Voice Agents
```

理由：

先懂 STT / TTS，再學 ASR、情緒、Voice Cloning，最後做 Agent 應用。

---

## 如果你想快速做作品

```text
1. Mistral AI Voxtral
↓
2. AI Voice Agents
↓
3. Speech Recognition with Python
↓
4. Mastering Voice AI
```

理由：

Mistral 和 Vapi 課比較容易快速做出可展示專案。

理由：
