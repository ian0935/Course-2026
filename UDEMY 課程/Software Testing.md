# 各課程詳細說明

## 1. The Complete 2026 Software Testing Bootcamp

這門是最像「QA 入門大全」的課。

內容包含：

* Software Testing 基礎
* Test scenario
* Test case
* Bug report
* Black-box testing
* White-box testing
* Agile testing
* Jira
* Zephyr Scale
* Trello
* Mobile testing
* API testing with Postman
* Performance testing with JMeter
* Selenium WebDriver
* SQL for testers
* Freelance testing
* QA CV / 面試 / 證照方向

它適合：

```text
完全沒學過 QA
想快速建立完整 QA 工作概念
```

推薦程度：**9 / 10**

---

## 2. MasterClass Software Testing with Jira & Agile - Be a QA Lead

這門最偏「公司實務流程」。

重點是：

* Manual testing
* Test case design
* Defect life cycle
* Jira
* Xray
* Agile Scrum
* SDLC
* Requirement analysis
* QA process
* Test planning
* QA Lead 角度

這門比較不是寫很多自動化程式，而是學：

```text
一個 QA 在真實軟體團隊裡怎麼工作
```

適合：

* 想理解 Jira
* 想學 Agile QA
* 想做 QA Lead
* 想補測試流程
* 想學專案中 QA 怎麼跟 PM / Dev 協作

推薦程度：**8.5 / 10**

---

## 3. Software Testing Masterclass (2026) - From Novice to Expert

這門比較精簡，約 10 小時左右。

內容包含：

* Software testing 基礎
* Black-box testing
* Boundary value analysis
* Equivalence partitioning
* Decision table
* State transition testing
* Web testing
* Mobile testing
* Backend testing
* Jira
* Xray
* TestRail
* Confluence
* Postman
* GitHub
* Miro
* Charles Proxy
* GenAI for test case generation

它適合：

```text
想用比較短的時間建立 QA 基礎
```

推薦程度：**8 / 10**

---

## 4. Practical Software Testing 2026 - Manual + JIRA + API + DB + Selenium

這門是最長、最重實作的傳統 QA 課。

內容包含：

* Manual testing
* Jira
* API testing
* Database testing
* SQL
* Python for automation
* Selenium with Python
* Data-driven framework
* BDD framework
* Automation interview questions

這門的核心路線是：

```text
Manual QA
↓
API Testing
↓
Database Testing
↓
Python
↓
Selenium Automation
```

適合：

* 想從 Manual QA 轉 Automation QA
* 想學 Selenium
* 想學 API + DB 測試
* 想準備 QA Automation 面試
* 想做大量實作

推薦程度：**9.5 / 10**

---

## 5. Gen AI for QA: Test Automation with Claude & GitHub Copilot

這門比較新，也比較短，重點是 AI 工具實作。

* Test plan generation

* AI-assisted test automation

它適合：

```text
已經會一點 QA 或程式
想快速把 AI 工具加進測試流程
```

推薦程度：**8.5 / 10**

它適合：

想自己做 QA Agent / 測試自動化工具

```text
AI Agent 工程

Software Testing
```

適合：

* 會 Python

## 7. GenAI & AI Agents for QA Automation | Copilot & Claude Code
