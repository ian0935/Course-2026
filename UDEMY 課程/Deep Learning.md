# 各課程定位

## 1. [PyTorch for Deep Learning Bootcamp](https://www.udemy.com/course/pytorch-for-deep-learning/?couponCode=KEEPLEARNING)

這門是 Zero to Mastery 的 PyTorch 主課。

Udemy 頁面顯示課程約 52 小時，內容包含 PyTorch fundamentals、tensors、training loop、classification、computer vision、transfer learning、deployment 等。

適合：

- 想學 PyTorch
    
- 想做 AI 工程作品集
    
- 想理解 training loop
    
- 想學 CNN / Transfer Learning
    
- 想部署 PyTorch 模型
    

推薦程度：**9.5 / 10**

---

## 2. [TensorFlow for Deep Learning Bootcamp](https://www.udemy.com/course/tensorflow-developer-certificate-machine-learning-zero-to-mastery/?couponCode=KEEPLEARNING)

這門是 Zero to Mastery 的 TensorFlow 主課。

Udemy 頁面顯示課程約 63 小時，涵蓋 TensorFlow 2、Computer Vision、CNN、NLP、Time Series Forecasting，以及可下載 notebooks 和 slides。

適合：

- 想學 TensorFlow / Keras
    
- 想做 CV、NLP、Time Series
    
- 想熟悉 TensorFlow 生態
    
- 想建立完整 DL 實作基礎
    

推薦程度：**9 / 10**

---

## 3. [Python for Computer Vision with OpenCV and Deep Learning](https://www.udemy.com/course/python-for-computer-vision-with-opencv-and-deep-learning/?couponCode=KEEPLEARNING)

這門是 Jose Portilla / Pierian Training 的 OpenCV 課。

Udemy 頁面顯示課程約 14 小時，涵蓋 NumPy、OpenCV 圖像處理、影片串流、邊緣/角點/格線偵測、人臉偵測、Watershed segmentation、物件追蹤與深度學習影像分類。

適合：

- Computer Vision 初學者
    
- 想學 OpenCV
    
- 想處理圖片與影片
    
- 想做人臉偵測、物件追蹤
    
- 想進 YOLO / OpenCV / ROS 影像前先打底
    

推薦程度：**9 / 10**

---

## 4. [Deep Learning: Advanced Computer Vision](https://www.udemy.com/course/advanced-computer-vision/?couponCode=KEEPLEARNING)

這門是 Lazy Programmer 的進階 CV 課。

Udemy 頁面顯示課程約 17 小時，涵蓋 VGG、ResNet、Inception、SSD、RetinaNet、Class Activation Maps、GANs、Neural Style Transfer、Object Localization 等。

適合：

- 已經懂 CNN
    
- 想學進階 CV 架構
    
- 想理解 Transfer Learning
    
- 想學 Object Detection
    
- 想學 GAN / Style Transfer
    

推薦程度：**9 / 10**

---

## 5. [A deep understanding of deep learning](https://www.udemy.com/course/deeplearning_x/?couponCode=KEEPLEARNING)

這門是 Mike X Cohen 的深度學習原理課。

它比較不是「快速做專案」，而是偏：

```text
深度學習到底怎麼運作？
為什麼模型會學？
Loss、Optimizer、Activation、Gradient 是什麼？
```

適合：

- 想真正理解 DL 原理
    
- 想補數學與 PyTorch 基礎
    
- 想讀 paper
    
- 想往 AI 研究或 ML Engineering 走
    
- 不想只會調 API
    

推薦程度：**9.5 / 10**

---

## 6. [Deep Learning A-Z](https://www.udemy.com/course/deeplearning/?couponCode=KEEPLEARNING)

這門是經典入門課，較偏快速總覽。

Udemy 頁面顯示它是 2026 版本，主題涵蓋 Deep Learning、AI in Python、AWS 等。

適合：

- 完全初學深度學習
    
- 想快速看過 ANN / CNN / RNN
    
- 想先建立直覺
    
- 想做一些經典小專案
    

推薦程度：**8 / 10**
