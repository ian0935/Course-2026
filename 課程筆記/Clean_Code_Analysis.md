# 2. 課程定位

Clean Code 並不是教你寫更多程式，而是教你如何寫出可維護、可閱讀、可擴充、可協作的程式碼。

在 AI Coding 時代，程式碼產生已經不困難，真正困難的是維護大型系統。

## 2.1 課程特色

- 可讀性（Readability）
- 命名規範（Naming Convention）
- 函式設計（Function Design）
- 模組化（Modularity）
- 重構（Refactoring）
- SOLID Principles
- Maintainability
- Technical Debt Management

## 2.2 適合對象

- AI Engineer
- Software Engineer
- Computer Science Students
- Full Stack Developers
- Backend Developers
- ERP 系統開發者
- Agent 系統開發者
