# 5. Important Keywords

### Voxtral
Mistral AI 的語音模型系列，可同時支援 STT 與 TTS。

### STT (Speech-to-Text)
將語音轉換成文字，例如即時字幕系統。

### TTS (Text-to-Speech)
將文字轉換成語音，例如 AI 語音助理。

### Voice Cloning
利用參考音訊複製說話者聲音。

### Real-Time Transcription
即時語音辨識技術。

### Streaming API
邊生成邊回傳結果，提高互動體驗。

### Function Calling
讓 AI 主動呼叫外部工具或函式。

### Tool Use
AI 能使用搜尋、資料庫、API等工具。

### AI Agent
具有自主決策能力的 AI 系統。

### Multi-Agent System
多個 AI Agent 協同合作完成任務。

### Agent Handoff
Agent 將工作交給其他專門 Agent。

### MCP (Model Context Protocol)
大型語言模型與外部工具溝通標準。

### Vision AI
理解圖片內容的 AI 技術。

### OCR
Optical Character Recognition，圖片轉文字。

### Document Understanding
理解文件與表單內容。

### Reasoning Model
具備較強推理能力的大語言模型。

### GDPR
歐盟個資保護法規。

### EU AI Act
全球首個大型 AI 法規框架。

### Data Sovereignty
資料主權，確保資料留在特定國家或區域。

### AI Compliance
AI 系統符合法規要求。
