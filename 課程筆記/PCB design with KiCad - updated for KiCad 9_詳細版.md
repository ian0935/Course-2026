# 2. 課程定位

## 2.1 課程特色

本課程是一套非常完整的 PCB 設計實戰課程。

與一般只教畫電路圖的課程不同，本課程從：

- 電子設計觀念
- 原理圖建立
- Footprint管理
- PCB Layout
- Design Rule Check
- Gerber輸出
- PCB打樣製造

一路教到產品真正送工廠生產。

最大特色是：

- 完整使用 KiCad 9
- 多個大型實作專案
- 包含 Git 版本管理
- 包含 Circuit Simulation
- 包含 2 Layer 與 4 Layer PCB
- 包含 Freerouting 自動佈線工具

## 2.2 適合對象

- 電機系學生
- 電子工程師
- 嵌入式系統開發者
- STM32 開發者
- ESP32 開發者
- ROS 機器人開發者
- Maker 與創客
- AIoT 開發者
