# 2. 課程定位

本課程屬於 Reinforcement Learning 進階課程，重點在於從傳統 RL 理論進入 Deep Reinforcement Learning，並使用 PyTorch 建立可實際運作的 AI Agent。

## 2.1 課程特色

- 強調理論與實作並重
- 使用最新 Gymnasium
- 使用 PyTorch 實作 Deep RL
- 包含 Atari Game AI 專案
- 涵蓋 DQN 與 A2C
- 介紹 Exploration vs Exploitation
- 介紹 Entropy Regularization
- 引入 Stable Baselines 3

## 2.2 適合對象

- AI Engineer
- Machine Learning Engineer
- Robotics Engineer
- Quantitative Finance Engineer
- Data Scientist
- AI Research Student
- 強化學習研究者
