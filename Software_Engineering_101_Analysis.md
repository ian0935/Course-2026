# 2. 課程定位

本課程屬於「軟體工程入門課」，重點不是教寫程式，而是教大型軟體系統如何規劃、設計、測試、部署與維護。

## 2.1 課程特色

- 完整介紹 Software Development Life Cycle (SDLC)
- Requirements 與 Specifications 分析
- WRSPM 模型
- Software Architecture Pattern
- Modularity、Coupling、Cohesion
- Testing Methodology
- Agile、SCRUM、Kanban
- AI 輔助軟體開發

## 2.2 適合對象

- Computer Science Students
- Junior Software Engineers
- AI Engineer 初學者
- 資訊工程系學生
- 專案管理與產品開發人員
