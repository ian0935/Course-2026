# 2. 數學基礎類課程

---

## Math 0-1: Linear Algebra for Data Science & Machine Learning

### 課程定位

AI 最重要的數學課之一。

主要教你：

- 向量
- 矩陣
- 線性變換
- Eigenvalue
- Eigenvector
- PCA 數學基礎

### 核心主題

- Vectors
- Matrices
- Matrix Multiplication
- Linear Transformations
- Systems of Equations
- Determinants
- Eigenvalues
- Eigenvectors
- PCA Foundation

### Keywords

```text
Linear Algebra
Vector
Matrix
Eigenvalue
Eigenvector
PCA
Linear Transformation
Determinant
Rank
Basis
```

### 適合程度

AI 必修。

---

## Master Linear Algebra: Theory and Implementation in Code

### 課程定位

Mike X Cohen 最強線代課之一。

不只教理論，還會：

- NumPy
- Python
- 視覺化
- Coding 實作

### 核心主題

- Matrix Factorization
- SVD
- Orthogonality
- PCA
- Covariance Matrix
- Projection

### Keywords

```text
SVD
Orthogonality
Projection
PCA
Covariance Matrix
Matrix Decomposition
Linear Independence
```

### 評價

比前面那門更深入。

建議兩門都上。

---

- Regression
- Statistical Modeling

### Keywords

```text
Probability
Bayes Theorem
Hypothesis Testing
ANOVA

Statistics
```

### 評價
